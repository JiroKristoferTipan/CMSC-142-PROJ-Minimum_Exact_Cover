# CMSC-142-PROJ-Minimum_Exact_Cover
- This code aims to show the difference between algorithm speeds, particularly comparing a brute force method and dynamic method. For this code, we will be using the `Minimum Exact Cover` as the topic to implement.


## Files
1. main.c
- main function to run, will check the time taken for both brute force and dynamic programming

2. data.h
- input will be read from here, ideally you want a huge set size/number of subsets to see difference in speed so manual user input is not very ideal

3. bruteforce.h 
- code for brute force method on minimum exact cover, checks through each subset by index to see which which once can be added

3. dynamic.h 
- code for dynamic method on minimum exact cover
